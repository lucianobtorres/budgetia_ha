# BudgetIA - Home Assistant Add-ons repository

This repository hosts the **BudgetIA** Add-on for Home Assistant.

## Installation

1. Go to **Settings** > **Add-ons** > **Add-on Store** in Home Assistant.
2. Click the three dots (top right) > **Repositories**.
3. Add this URL: `https://github.com/lucianobtorres/budgetia_ha`
4. Find **BudgetIA (Monorepo)** in the list and install it.

## Structure

- `budgetia/`: Contains the full application source code and add-on configuration.
